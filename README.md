# 🎨 **Color Palette Generator v2.0.0**

先進的なAI駆動カラーパレット生成・管理システム

---

## ✨ **主な機能**

### 🎯 **マルチレベル対応**
- **🧒 Elementary Mode**: 子供向けの大きく楽しいインターフェース
- **👨‍💼 Middle School Mode**: 大人向けの高機能プロフェッショナルツール
- **🔬 Advanced Mode**: 科学的色彩理論に基づく精密機能

### 🔬 **高度な色処理**
- **HSL色空間**: 直感的な色相・彩度・明度調整
- **LAB色空間** ⭐: 人間の視覚により近い科学的色調整
- **LCH色空間** ⭐: 明度・彩度・色相の円筒座標系での調整
- **Delta E 2000計算** ⭐: 最新の色差測定アルゴリズム
- **リアルタイムプレビュー**: 調整内容を即座に確認

### ♿ **アクセシビリティ機能**
- **WCAG 2.1準拠チェック**: AA/AAA基準に対応
- **コントラスト比自動計算**: 4.5:1（AA）、7:1（AAA）基準
- **色覚多様性テスト**: 1型・2型・3型色覚、全色盲シミュレーション
- **改善提案機能**: 自動的な色調整提案

### 💼 **プロジェクト管理** ⭐
- **複数パレット管理**: プロジェクト単位での体系的整理
- **タグ機能**: カテゴリ別分類・検索
- **バージョン管理**: 変更履歴の追跡・復元
- **チーム共有**: プロジェクトの共有・協業機能

### ⚡ **バッチ処理** ⭐
- **複数画像対応**: 最大20画像の一括処理
- **3つのアルゴリズム**:
  - **K-means**: 高品質な色抽出（推奨）
  - **Median-Cut**: バランスの取れた抽出
  - **Octree**: 高速処理
- **並行解析**: WCAG・色覚テストも同時実行
- **ZIP一括エクスポート**: 結果の統合出力

### 📚 **教育機能** ⭐
- **色理論パネル**: インタラクティブ色彩学習
- **色相環ディスプレイ**: 視覚的な色関係理解
- **年齢別カリキュラム**: 小学生〜プロまで対応

## 🚀 **セットアップ手順**

### 1️⃣ **依存関係のインストール**

```bash
# プロジェクトディレクトリに移動
cd C:\Users\karum\Downloads\color-palette-generator

# 依存関係をインストール
npm install
```

**📦 新機能で追加されたライブラリ:**
- `culori`: 高度な色空間変換（LAB/LCH対応）
- `color-blind`: 色覚異常シミュレーション
- `jszip`: バッチ処理結果のZIP出力
- `chroma-js`: 基本的な色変換・操作
- `colorthief`: 画像からの色抽出

### 2️⃣ **開発サーバーの起動**

```bash
npm run dev
```

ブラウザで `http://localhost:3000` にアクセス

### 3️⃣ **本番ビルド**

```bash
npm run build
npm start
```

## 🎨 **使用方法**

### 🎯 **基本ワークフロー**

#### 1. **🎭 モード選択**
   - **🧒 Elementary**: 大きなボタン、シンプルな操作
   - **👨‍💼 Middle School**: バランスの取れた機能性
   - **🔬 Advanced**: プロフェッショナル向け全機能

#### 2. **📷 画像アップロード**
   - 対応形式: JPG, PNG, GIF, WebP
   - ドラッグ&ドロップ対応
   - 複数画像の一括処理（バッチモード）

#### 3. **🎨 パレット生成**
   ```
   例: 抽出色数選択
   ┌─────────────────┐
   │ 3色 4色 5色 ⭐  │
   │ 6色 7色 8色 9色 │
   └─────────────────┘
   ```
   - K-meansアルゴリズム推奨
   - リアルタイムプレビュー

#### 4. **🔬 精密調整**（Advanced Mode）
   ```
   HSL色空間: H(0-360°), S(0-100%), L(0-100%)
   LAB色空間: L(0-100), a(-128~127), b(-128~127) ⭐
   LCH色空間: L(0-100), C(0-150), H(0-360°) ⭐
   ```

#### 5. **♿ アクセシビリティチェック**
   ```
   WCAG基準:
   ┌──────────────────────────────┐
   │ AA標準: 4.5:1 (通常) 3:1 (大)│
   │ AAA厳格: 7:1 (通常) 4.5:1 (大)│
   └──────────────────────────────┘
   ```

#### 6. **💾 保存・エクスポート**
   - **プロジェクト保存**: ローカルストレージ + クラウド同期
   - **形式選択**: JSON / CSS / SCSS / Adobe ASE
   - **解析結果付き**: WCAG + 色覚テスト結果も含む

## 🎓 **色彩理論について**

### 🔬 **LAB色空間の優位性**
LAB色空間は人間の視覚システムに基づいて設計されており、数値的な距離が知覚的な色の違いに比例します。

```
L (明度): 0-100  → 黒から白への明るさ
a (緑-赤軸): -128 to 127 → 緑から赤への色相
b (青-黄軸): -128 to 127 → 青から黄への色相
```

### 📐 **Delta E 2000の重要性**
```
Delta E値の解釈:
┌─────────┬──────────────────────────────┐
│ 0-1     │ ✅ 人間には同一色に見える      │
│ 1-2     │ 🔍 訓練された目でも困難        │
│ 2-10    │ 👁️ デザイナーが識別可能        │
│ 11-49   │ 🌈 明確に異なる色として認識    │
│ 50-100  │ 🎨 完全に異なる色              │
└─────────┴──────────────────────────────┘
```

### ♿ **アクセシビリティの科学的根拠**
```
WCAG 2.1基準:
┌─────────┬──────────────────────────────┐
│ AA基準  │ 通常文字 4.5:1以上 大文字 3:1以上│
│ AAA基準 │ 通常文字 7:1以上 大文字 4.5:1以上│
└─────────┴──────────────────────────────┘

対象人口:
• 軽度視力障害者: 約15%（AA基準対応）
• 重度視力障害者: 約3%（AAA基準対応）
• 色覚多様性: 男性8%、女性0.5%
```
